*node.js 관련 
먼저 node.js를 설치한 후에 아래와 같이 필요한 모듈을 설치할 수 있다. 
https://nodejs.org/en/

$ npm install express

설치가 필요한 패키지를 아래와 같이 생성한다.
파일이름은 package.json 을 사용한다. 
{
  "name": "express-tutorial",
  "version": "1.0.0",
  "dependencies": 
  {
    "express": "~4.13.1",
    "ejs": "~2.4.1"    
  }
}

위의 셋팅을 아래와 같이 실행한다. 
npm install

버전이 낮아서 경고가 출력되면 아래와 같이 업데이트한다.
npm audit fix --force
